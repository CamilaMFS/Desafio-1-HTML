História da Computação

Voltar

A história da computação remonta a milhares de anos, desde os primeiros dispositivos mecânicos, como o ábaco, até os computadores modernos. Os primeiros computadores eram máquinas enormes, como o ENIAC, desenvolvido nos anos 1940, que ocupavam salas inteiras e realizavam cálculos complexos. Ao longo do tempo, a computação evoluiu, tornando-se cada vez mais compacta, poderosa e acessível.

Grandes Nomes

Voltar

Alguns dos grandes nomes na história da computação incluem Charles Babbage, considerado o "pai do computador", Ada Lovelace, uma das primeiras programadoras, Alan Turing, conhecido por seus trabalhos fundamentais em computação e que desempenhou um papel crucial na Segunda Guerra Mundial decifrando códigos,entre outros.

Linguagens de Programação

Voltar

Linguagens de programação são conjuntos de instruções utilizadas para criar programas e aplicativos de computador. Existem linguagens de programação para diferentes propósitos, como desenvolvimento de software para servidores (server side) e para interação com o usuário (client side). HTML não é considerada uma linguagem de programação, mas sim uma linguagem de marcação usada para criar páginas da web.

HTML 5

Voltar

HTML 5 é a última versão da linguagem de marcação HTML, amplamente utilizada para criar páginas da web. Ela define a estrutura básica de uma página web, incluindo tags para organização e formatação do conteúdo, como cabeçalhos, parágrafos, listas e links. A linguagem também oferece atributos para controlar o comportamento e a aparência dos elementos na página.

Tags

Tags HTML são elementos usados para estruturar e formatar o conteúdo de uma página da web. Elas são escritas entre colchetes angulares e podem ter atributos para controlar seu comportamento e aparência.

Atributos

Atributos HTML fornecem informações adicionais sobre um elemento e podem modificar seu comportamento ou aparência. Eles são especificados dentro da tag de um elemento e consistem em um nome e um valor.

Textos

Textos em HTML são exibidos na página da web como conteúdo legível pelo usuário. Eles podem ser formatados usando tags como < p> para parágrafos, < h1> a < h6> para cabeçalhos de diferentes níveis, < strong> entre outros.

Listas

Listas HTML são usadas para organizar itens relacionados em uma ordem específica. Existem dois tipos principais de listas: listas ordenadas < ol> e listas não ordenadas < ul>. Cada item da lista é definido dentro de uma tag < li>.

Links

Links HTML (< a>) são usados para criar conexões entre diferentes páginas da web. Eles permitem que os usuários cliquem em um texto ou imagem para serem redirecionados para outra página da web. O atributo href é usado para especificar o URL de destino do link.
